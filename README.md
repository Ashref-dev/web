# RecipeShare

![RecipeShare](/home.png)

A neo-brutalist recipe sharing platform where users can discover, create, and share delicious recipes.

## Quick Start

```bash
docker-compose up
```

Wait a few seconds for MongoDB In-Memory to download and initialize, then open `http://localhost:3000`.

## What's Included

- **Frontend**: Next.js 16, React, Tailwind CSS
- **Backend**: Express.js, TypeScript, MongoDB In-Memory
- **Features**: User authentication, recipe CRUD, comments, favorites, search, pagination
- **Design**: Neo-brutalist with yellow accents
